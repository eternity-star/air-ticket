# air-ticket

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 前端需要启动本地服务；进入到server文件夹路径下，输入下面的命令
```
node index
```

### 启动后台mysql服务
```
net start mysql
```
### 关闭后台mysql服务
```
net stop mysql
```

### Compiles and minifies for production
```
npm run build
```
